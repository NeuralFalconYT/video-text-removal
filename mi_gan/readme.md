

## 🙏 Credits & Acknowledgements

### 🔹 IOPaint

* GitHub: [https://github.com/Sanster/IOPaint](https://github.com/Sanster/IOPaint)
* Author: Sanster

IOPaint provides the **inpainting pipeline, preprocessing logic, and MI-GAN integration** used in this project.
Parts of this codebase are adapted from IOPaint to enable standalone MI-GAN usage for video processing.

### 🔹 MI-GAN (Masked Image Generation GAN)

* GitHub: [https://github.com/Picsart-AI-Research/MI-GAN](https://github.com/Picsart-AI-Research/MI-GAN)
* Authors: Picsart AI Research

MI-GAN is the **core inpainting model** used for removing text regions and restoring background content.
This project uses the official MI-GAN model weights and inference logic.
